= Configuring your Kubernetes API server
include::modules/common-attributes.adoc[]
:context: configuring-apiserver

toc::[]

{product-title} deploys a Kubernetes API server component on the master. It
serves as the front-end for the control plane and validates and configures data
for Pods, Services, and ReplicationControllers.

Cluster administrators can customize this component by modifying the `APIServer`
configuration resource.
